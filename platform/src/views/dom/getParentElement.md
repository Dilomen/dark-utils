# getParentElement

> 获取该元素的父元素

| 参数                                     | 返回 | 描述 |
| ---------------------------------------- | ---- | ---- |
| (**ele**: Element, **class**: className) | void |      |

## 使用

```js
import { addClass } from 'serein-utils';
addClass(element, className);
```

## 例子

```js
import { addClass } from 'serein-utils';
const _oDiv = document.getElementById('app');
addClass(_oDiv, 'wrap');
```
